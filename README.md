# 💸 App Gastos

Aplicación web para la gestión de gastos personales, ingresos y presupuestos, diseñada especialmente para jóvenes que comienzan a vivir de forma independiente.

## 🚀 Descripción

App Gastos permite a los usuarios llevar un control claro y organizado de sus finanzas personales. La aplicación facilita el registro de ingresos y gastos, la creación de presupuestos y el análisis visual del dinero para tomar mejores decisiones.

Está pensada para personas entre 20 y 25 años que se mudan a una nueva ciudad (como Buenos Aires) y necesitan organizar sus gastos de forma simple.

---

## 🧩 Funcionalidades principales

### 👤 Autenticación de usuarios
- Registro e inicio de sesión
- Manejo de sesiones con Firebase Authentication

### 💰 Gestión de gastos
- Carga de gastos
- Edición y eliminación
- Historial de gastos

### 📊 Ingresos
- Registro de ingresos totales
- Visualización general del dinero disponible

### 📉 Presupuestos
- Creación de presupuestos por categoría
- Control de gastos vs presupuesto
- Alertas cuando se supera un presupuesto

### 📈 Gráficos
- Visualización de datos financieros
- Análisis de gastos por categoría

### 📅 Calendario de pagos
- Organización de pagos futuros
- Visualización de fechas importantes

### 🔔 Notificaciones
- Indicadores de eventos importantes (como presupuestos excedidos)

### 👤 Perfil de usuario
- Visualización de datos del usuario
- Personalización básica

## 🧠 Objetivo del proyecto

Este proyecto surge como solución a una problemática real: la dificultad de gestionar el dinero al comenzar una vida independiente. La app busca simplificar la organización financiera y fomentar mejores hábitos de consumo.
